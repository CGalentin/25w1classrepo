1. Understanding the DOM

The DOM (Document Object Model) is a programming interface that represents the structure of HTML documents as a tree of objects. Every HTML element becomes a node in this tree.

DOM Tree Structure:

  • Document
    • HTML
      • HEAD (title, meta, style...)
      • BODY
        • DIV (containers)
        • P (paragraphs)
        • BUTTON (interactive elements)
