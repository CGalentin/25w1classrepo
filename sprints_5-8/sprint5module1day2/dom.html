1. Setting Text with .textContent

â–¼

The .textContent property allows you to get or set the text content of an element, stripping away any HTML tags.

// Getting text content const element = document.getElementById('myElement'); console.log(element.textContent); // Returns the text // Setting text content element.textContent = 'Hello, World!';
Note: .textContent is safer than .innerHTML when dealing with user input as it prevents XSS attacks by treating content as plain text.

3. Setting Inline Styles with .style

â–¼

The .style property allows you to set inline CSS styles directly on elements.

// Setting individual styles element.style.color = 'red'; element.style.backgroundColor = 'yellow'; element.style.fontSize = '20px'; // Setting multiple styles with cssText element.style.cssText = 'color: blue; background: yellow; font-size: 18px;'; // Getting computed styles const computedStyle = getComputedStyle(element); console.log(computedStyle.color);
Warning: CSS property names in JavaScript use camelCase (backgroundColor) instead of kebab-case (background-color).
